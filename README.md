# Ola Ride Cancellation Analysis | Power BI & SQL Project

## Project Overview
Developed an interactive Power BI dashboard to analyze Ola ride booking and cancellation trends.  
The project focuses on identifying cancellation patterns, revenue insights, customer ratings, and vehicle performance to support data-driven business decisions.

---

## Business Problem
Ride cancellations negatively impact customer experience and operational efficiency.  
This project aims to identify the major factors contributing to ride cancellations and provide actionable insights to improve overall ride success rates.

---

## Objectives
- Analyze ride cancellation trends
- Identify high-cancellation vehicle categories
- Evaluate revenue performance
- Study customer ratings and booking behavior
- Generate business insights using data visualization

---

## Tools & Technologies Used

- **Power BI** → Dashboard creation & visualization
- **SQL** → Data querying and analysis
- **Excel** → Data cleaning & preprocessing

---

## Dashboard Features

### Vehicle Type Analysis
- Comparison of ride bookings by vehicle category
- Vehicle-wise cancellation trends

### Revenue Analysis
- Revenue generated across different ride types
- Revenue trend visualization

### Cancellation Analysis
- Major cancellation reasons
- Cancellation percentage breakdown
- Trend analysis

### Customer Ratings Analysis
- Driver and customer rating insights
- Service quality comparison

### Interactive Dashboard
- Multi-page navigation
- Dynamic filters and slicers
- KPI cards and visual analytics

---

## Key Insights
- Identified major causes of ride cancellations
- Observed cancellation trends across vehicle categories
- Analyzed customer satisfaction using ratings
- Suggested strategies to reduce cancellation rates and improve ride completion
